# Атланты — веб-арбитр PvP-дуэлей

Веб-приложение для игры в «Атлантов» физическими картами один на один: каждый игрок
открывает страницу на своём телефоне, приложение ведёт счёт и следит за правилами.

## Как играть

1. Игрок 1 открывает сайт, вводит имя, жмёт **«Создать бой»** — получает 4-значный код.
2. Игрок 2 на своём телефоне вводит этот код — бой начинается автоматически.
3. На своём ходу игрок тапает карты в руке (они разыгрываются в противника или на себя
   по правилам карты) и жмёт **«Конец хода»**.
4. Победа — когда HP противника падает до нуля. Кнопка «Реванш» начинает новый бой.

## Правила (перенесены из Unity-клиента игры)

- Старт: 90 HP, 3 энергии за ход, рука 5 карт (значения из `UserStateHolder.GetStartStats`).
- Урон = базовый + сила атакующего. Блок гасит урон первым, остаток идёт в HP.
- Блок = базовый + ловкость. Сгорает в начале **своего** следующего хода.
- Энергия восстанавливается до максимума каждый ход (не копится).
- Сила и ловкость — постоянные баффы до конца боя.
- Карта с «Изгнанием» (Disappear) уходит из боя насовсем, не в сброс.
- Пустая колода перетасовывается из сброса.
- Домашнее правило PvP: первый игрок на первом ходу получает на 1 энергию меньше
  (компенсация преимущества первого хода). Отключается в `engine.js` → `firstTurnHandicap`.

## Запуск

```
node server.js        # локально, порт 3000
node test-engine.js   # 17 автотестов движка
```

Зависимостей нет — чистый Node.js. Деплой на Railway: подключить репозиторий,
порт берётся из `process.env.PORT` автоматически.

## Структура

- `engine.js` — боевой движок, правила 1:1 из Unity-кода (`Battle/Commands/*`)
- `server.js` — комнаты, API, раздача статики; состояние в памяти, TTL комнаты 6 часов
- `public/index.html` — мобильный клиент (лобби + экран боя), опрос сервера раз в 1.2 с
- `cards.json` — **ДЕМО-колода**. Формат полей повторяет `CardGamePlayData` Unity-клиента;
  после выгрузки реальных конфигов (`cardsAttack`, `cardsDefend`, `cardsMagic` из
  Unity Remote Config проекта Atlanteans) файл заменяется на настоящие карты без правки кода.

## Что осталось до «настоящих Атлантов»

1. Достать реальные конфиги карт из Unity Remote Config (нужен доступ к Unity Dashboard
   организации r-proc, проект Atlanteans, либо дамп из запущенного клиента).
2. Заменить `cards.json`, проверить эффекты CopyCard/ThrowCard на реальных картах.
3. Выбор колоды перед боем (сейчас обе колоды одинаковые стартовые).
4. Арты карт из атласов `Assets/_SPAR/UI/Cards` (Attack1-28, Defend1-28, Magic1-28).
